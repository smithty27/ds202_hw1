  1. Install git.

    • If you are on Windows operating system:
      1. Install Git for Windows. In particular, we will need to use the Git Bash (not the Git GUI).
      2. After installation, set up Rstudio to use Git bash as the shell for the Terminal: Open RStudio, click ToolsGlobal OptionsTerminal; under Shells, select Git Bash; click OK.
      3. In the main menu of RStudio: click ToolsTerminalNew Terminal. If you have a $ before your prompt, then you should not need to do anything.
    • If you are on Linux or macOS, Git has probably been correctly configured by default, so nothing needs to be installed. Just click the Terminal tab (above the R Console).

  2. What is the version of your Git? To find out, go to the Terminal tab in RStudio (right to the Console tab), and type in the following command. If it works, this shows that the setup of your terminal and Git is correct. Then explore the bash command line and Git.
